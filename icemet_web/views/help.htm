Help

Database Fields

Particles

  • ID: Database entry ID.
  • DateTime: Date and time of the measurement.
  • Sensor: Sensor ID.
  • Frame: Image frame number.
  • X, Y, Z: Particle centroid coordinates in the image (m).
  • EquivDiam: Equivalent Diameter (m).
  • EquivDiamCorr: Used Equivalent Diameter correction factor.
  • Circularity: Heywood Circularity Factor. Lower = more circular.
  • DynRange: Dynamic range of the particle image.
  • EffPxSz: Effective pixel size of the particle image (m/px).
  • SubX, SubY, SubW, SubH: Particle segment rectangle in the Preview image (px).

Statistics

  • ID: Database entry ID.
  • DateTime: Date and time of the measurement.
  • LWC: Liquid Water Content (g/m3).
  • MVD: Median Volume Diameter (m).
  • Conc: Particle concentration (#/m3).
  • Frames: Number of images used in the measurement.
  • Particles: Total number of particles found.
